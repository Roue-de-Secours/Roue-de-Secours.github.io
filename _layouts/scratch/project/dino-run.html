Dino Run

Le dinosaure court pour survivre ! Aide le à éviter tout les rochers

Résumé

Appuie sur n'importe quel touche du clavier pour faire sauter le dinosaure afin qu'il ne trébuche pas sur les rochers.

Démo

Toggle édition Re-charger le projet

Réalisation

Identification des objectifs du jeu

On va se poser la question de qu'est ce qu'on veut ?

( déjà répondu dans le résumé )

Identification des lutins

On va se poser la question de quoi avons nous besoin et que doit-il faire?

Dans notre cas j'ai choisi d'en utiliser 2 :
  • Dinosaure
  • Rocher

Programmation des lutins

Dinosaure
Sauter

Comme le saut monte et descend toujours à la même hauteur, il suffit de jouer avec 2 boucles, une qui va le faire monter, puis une qui va le faire descendre.

Collisions avec le rocher

Le jeu s'interrompt quand le dinosaure heurte (touche) un rocher

Rocher
Déplacement

Dans ce jeu, seul le rocher se déplace.
Il suffit de le déplacer vers la gauche.
N'oublie pas de le téléporter à droite quand le rocher est "trop à gauche"
